## 🧠 Projects Overview

### 🤖 AI Chatbot (Java)
- A simple GUI-based chatbot built using Java
- Uses predefined FAQ responses
- Demonstrates:
  - File handling
  - GUI development (Swing)
  - Basic NLP logic

---

### 📊 Student Grade Tracker
- Console-based Java application
- Allows adding and managing student grades
- Calculates:
  - Average
  - Highest score
  - Lowest score

## ▶️ How to Run

### 1. Clone the Repository
```bash
git clone https://github.com/bhaviniawasthi1/codealpha_tasks.git
cd codealpha_tasks
````

### 2. Compile Java Files

```bash
javac *.java
```

### 3. Run Program

```bash
java ClassName
```

> Example:

```bash
java ChatbotGUI
```
